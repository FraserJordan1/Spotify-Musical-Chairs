Detailed Description

ECE Lab 4 Assignment, Part C @detail University of Washington ECE 474 Acknowledgments: LCD was inspired by: https://www.instructables.com/Arduino-Interfacing-With-LCD-Without-Potentiometer/ RFID was inspired by: https://www.instructables.com/Interfacing-RFID-RC522-With-Arduino-MEGA-a-Simple-/.

Author
Jordan Fraser, Nina Okubo
Date
August 18 23, 2023

This program continuously flashes an external LED (Task 1), and waits for an RFID card to be scanned, then sends the scanned card's Spotify code to the Serial to play the song/album on Spotify. Also recieves information about the current song playing on Spotify and displays it on an LCD screen.

void detectCard ( void *  pvParameters)

Waits for the RFID scanner to scan a card.

Author
Jordan Fraser, Nina Okubo

Constantly checks if a card has been detected. If a card is detected, gets the serial number value and checks that the card is different from the previous scan If the card is different, then find the associated Spotify code and send the code to Serial. If the card is not recognized then nothing happens Acknowledgments: Inspired by https://www.instructables.com/Interfacing-RFID-RC522-With-Arduino-MEGA-a-Simple-/
